<h1>기능 목록</h1>

1. main
 - 메인.
 
2. get_input
 - 자동차 이름과 이동 횟수를 입력받는다.
 
3. racing
 - 자동차들의 레이싱을 진행한다.

4. print_status
 - 현재 자동차들의 경주 상황을 출력한다.

5. gostop
 - 자동차가 이동할 지 여부를 정해준다.
 
6. random_value
 - 전진하는 조건에 해당하는 0~9사이의 임의의 값을 리턴한다.

7. find_winner
 - 승자를 찾아 배열에 기록한다.

8. find_max_distance
 - 가장 멀리 진행한 자동차의 이동거리를 찾는다.

9. print_winner
 - 우승자 목록을 출력한다.
